# Finance Tracker Bot

Telegram бот для учета финансов с поддержкой нескольких кошельков, категорий расходов/доходов и статистики.

## Возможности

- 📊 Учет доходов и расходов по категориям
- 💳 Управление несколькими кошельками
- 📈 Детальная статистика и аналитика
- 🌐 Поддержка русского и узбекского языков
- 💱 Поддержка различных валют
- 🛡️ Админ панель для управления пользователями
- 🔔 Система напоминаний (APScheduler)

## Установка и запуск

### Локальная установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/your-username/finance-bot.git
cd finance-bot
```

2. Создайте виртуальное окружение:
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate  # Windows
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

4. Создайте файл `.env` и добавьте токен бота:
```env
BOT_TOKEN=your_bot_token_here
ADMIN_ID=1265652628
```

5. Запустите бота:
```bash
python bot/main.py
```

### Деплой на Railway

1. Форкните этот репозиторий
2. Зарегистрируйтесь на [Railway](https://railway.app)
3. Создайте новый проект и подключите GitHub репозиторий
4. В настройках проекта добавьте переменную окружения `BOT_TOKEN` с токеном вашего бота
5. Railway автоматически развернет проект

### Деплой на других платформах

Бот совместим с любыми платформами, поддерживающими Python 3.11+:
- Heroku
- PythonAnywhere
- VPS/VDS
- Docker

## Использование

### Для пользователей

1. Начните чат с ботом командой `/start`
2. Выберите язык и валюту в настройках
3. Создайте кошельки для разных целей
4. Добавляйте доходы и расходы, выбирая соответствующие категории
5. Просматривайте статистику и историю операций

### Для администратора

1. Используйте команду `/admin` для доступа к админ панели
2. Просматривайте список всех пользователей
3. Блокируйте/разблокируйте пользователей
4. Просматривайте общую статистику и транзакции

## Технологии

- **Python 3.11** - Язык программирования
- **aiogram 3.x** - Telegram Bot API framework
- **SQLite** - База данных
- **APScheduler** - Планировщик задач
- **Railway** - Платформа для деплоя

## Безопасность

- Все токены хранятся в переменных окружения
- Админ панель защищена ID администратора
- Данные пользователей защищены
- Используются безопасные методы работы с базой данных
